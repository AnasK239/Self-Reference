Default behavoiur is returning a hash code for the unique identifier so it must be overriden to be useful

its good practice to add @Override when overriding something.

Usually the toString method is as follows

public String toString() {
    return getClass().getName() + "@" + Integer.toHexString(hashCode());
}

We Override it to :

@Override
    public String toString()
    {
        return (String characteristics);
    }

This way if you call sout ( Object ) >>> this will return the actual string of text intended not its hash
